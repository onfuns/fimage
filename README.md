### 开发

```
npm run start
npm run electron
```

> 注意：electron 在 windows 系统调试开发时不能用管理员模式运行，否则会禁用拖拽文件功能。

### 编译

```
npm run build:mac
npm run build:win32
npm run build:win64
```

### 打包 all

```
npm run build:package
```
